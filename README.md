### 编译

编译文件存放在 _out
```
make
```

### 安装

安装路径：/usr/ 
```
make ; make install
```
如果没有权限，使用
```
make ; sudo make install
```
如果想手动指定安装路径，可使用：
```
make DESTDIR=/outpath install
```
